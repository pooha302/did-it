# Language Preference Rule

## Rules
- 사용자에 대한 모든 응답은 반드시 **한국어**로 작성해야 합니다.
- 이는 설명, 요약 및 모든 대화 내용에 적용됩니다.
- 코드 주석이나 기술 용어는 관례에 따라 영어를 사용할 수 있지만, 그 외 모든 설명은 한국어여야 합니다.

## Examples
### ✅ 좋음
사용자 요청에 따라 기능을 수정했습니다.

### ❌ 나쁨
I updated the feature as requested.
